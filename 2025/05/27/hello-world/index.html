源文件备份

  • 切记备份好本地的源文件，尤其是 Markdown 文件，其他配置一旦丢失则无法正常写博客，需要从头开始设置
  • 建议使用 GitHub 同一个仓库备份
  • 建议每当有一些改动就备份一次，或者每日备份一次
  • 更多用法请查看 Git 文档
